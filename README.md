# WBS Tool

Webbasiertes, datenbankgestütztes WBS-Tool zur Verwaltung, Bearbeitung, Versionierung und Auswertung von Projektstrukturen.

## Ziel

Das Tool soll eine Work Breakdown Structure (WBS) zentral verwalten und später um weitere Module wie Auslastung, Kompetenzen und Ressourcenplanung erweiterbar sein.

## Geplanter MVP

- Projektverwaltung
- WBS manuell anlegen
- WBS als Baumstruktur anzeigen
- WBS bearbeiten und verschieben
- Excel-Import
- Rollen und Rechte
- Dashboard
- Audit-Log
- Versionierung
- Export nach Excel/CSV

## Projektstruktur

- `docs` – Dokumentation
- `src/backend` – Backend-Anwendung
- `src/frontend` – Frontend-Anwendung
- `scripts` – Hilfsskripte
- `infra` – Infrastruktur / Deployment-Vorbereitung
